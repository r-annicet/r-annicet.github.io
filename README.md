# Personal Academic Website

This project is a modern static personal website designed for a PhD student applying to postdoctoral positions. It is ready to host on GitHub Pages.

## Files

- `index.html`: page structure
- `styles.css`: visual design and responsive layout
- `site-data.js`: all personal content in one place
- `script.js`: renders the content into the page

## First edits to make

Open `site-data.js` and update:

- your name
- role and institution
- email and location
- summary and research focus
- publications
- experience
- links to Google Scholar, GitHub, CV, and any lab page

## How GitHub Pages works

GitHub Pages is a free static website host provided by GitHub.

For a personal site, the most common setup is:

1. Create a GitHub repository named `yourusername.github.io`
2. Upload the files from this folder into that repository
3. In GitHub, open the repository settings and find `Pages`
4. Set the source to deploy from the main branch
5. Your website will appear at `https://yourusername.github.io/`

If you instead use a normal repository name such as `profile`, the site URL usually becomes:

`https://yourusername.github.io/profile/`

That is called a project site. For a personal academic homepage, the `yourusername.github.io` repository is usually the better choice.

## Publish steps

1. Create a GitHub account if you do not already have one
2. Create a new repository named `yourusername.github.io`
3. Add these files to that repository
4. Commit and push
5. Wait a minute or two for GitHub Pages to build
6. Open `https://yourusername.github.io/`
